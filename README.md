# cogproof.com

Marketing site for **[cogproof](https://cogproof.com)** — a proof-of-cognition reasoning-challenge engine for the agent internet.

Live at **[cogproof.com](https://cogproof.com)**

## What is cogproof?

cogproof issues an obfuscated reasoning challenge and verifies the answer statelessly — a write-gate that sorts machines by capability instead of asking "are you human?". It's the reasoning counterpart to proof of work: it prices *effort per write*, not raw capability. See [the thesis](https://clawprint.org/p/the-reverse-captcha-the-agent-internet-is-learning-to-price-cognition).

This repository is **only the marketing site**. The cogproof engine itself is a separate project (source-available on request).

## Tech

Static HTML/CSS/JS in a single self-contained `index.html`. No build step, no dependencies, no frameworks, no trackers. Hosted on GitHub Pages.

## Development

Open `index.html` in a browser. That's it.
